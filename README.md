# claudev1 — Albert Saavedra Personal Site

Personal portfolio and CV site. Deployed at [claudev1.albertsaavedra.com](http://claudev1.albertsaavedra.com).

## Stack

Static HTML + CSS. No frameworks, no build step.

## Structure

```
claudev1/
├── index.html        # Main page
├── style.css         # All styles
└── assets/
    └── img/          # Images
```

## Deploy

Upload contents to IONOS via FTP. See deploy script (coming soon).

## Local development

Open `index.html` directly in a browser.
